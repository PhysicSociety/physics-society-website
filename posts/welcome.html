Welcome to the Physics Society blog

Team • 2025-09-06

Here’s how anyone in the club can publish a post in minutes—no command line needed. If you get stuck, email us at physicsociety2025@gmail.com.

Quick start (GitHub website, easiest)

  1. Open our GitHub repo in your browser.
  2. Click the posts folder.
  3. Click Add file → Create new file.
  4. In the filename box, type: posts/YYYY-MM-DD-short-title.html
    Example: posts/2025-09-07-quantum-rumors.html
  5. Paste the Post Template below into the editor.
  6. Change the Title, Author, Date, and write your post.
  7. Scroll down and click Commit changes.
  8. Your post will be live at: /physics-society-website/posts/YYYY-MM-DD-short-title.html in ~1 minute.
  9. Last step: add a card for your post to posts.html (instructions below).

Add images (optional)

  1. In the repo, click assets → Add file → Upload files.
  2. Upload your image(s). Use descriptive names (e.g., lattice-defect-sem.png).
  3. In your post, reference them like:
    <img src="../assets/lattice-defect-sem.png" alt="SEM of lattice defect" style="max-width:100%;height:auto;border-radius:8px;">

Add your post to the listing (posts.html)

  1. Open posts.html → Edit.
  2. Inside the <div class="grid"> ... </div>, copy one of the existing cards and edit it, e.g.: 
    <div class="card">
  <h3><a href="posts/2025-09-07-quantum-rumors.html">Quantum Rumors: What’s Real?</a></h3>
  <small>By Your Name • 2025-09-07</small>
  <p>A quick tour of what’s hype versus hope in quantum tech.</p>
</div>
  3. Commit changes. Your post now appears on the Blog page.

Writing tips

  • Length: 500–900 words works great.
  • Audience: smart undergrads & curious non-majors—explain jargon once.
  • Images: add alt text; use your own figures or properly credited ones.
  • Citations: link sources; avoid copy-paste from elsewhere.
  • Tone: skeptical, clear, and fun—evidence over vibes.

Advanced (optional, for git users)

  1. git clone <repo-url>
  2. Add your file in posts/ and images in assets/.
  3. git checkout -b post/<short-title>
  4. git add . && git commit -m "Add post: <title>"
  5. git push -u origin HEAD then open a Pull Request.
