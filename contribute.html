How to Publish a Blog Post

Follow these steps to write and publish your own article for the Physics Society blog. No coding experience required!

Step 1: Open the Repo

Go to our GitHub repository in your browser and click the posts folder.

Step 2: Create a New File

  1. Click Add file → Create new file.
  2. Enter a filename like: posts/2025-09-07-your-title.html

Step 3: Paste Template & Write

Paste this template and replace the title, author, and content:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width,initial-scale=1">
  <title>POST TITLE • Physics Society</title>
  <link rel="icon" type="image/png" href="../assets/favicon.png">
  <link rel="stylesheet" href="../styles.css?v=8">
</head>
<body>
<header class="starfield">... (nav) ...</header>
<main>
  <h1>POST TITLE</h1>
  <small>Your Name • YYYY-MM-DD</small>
  <hr>
  <p>Write your post here.</p>
</main>
<footer>...</footer>
</body>
</html>

Step 4: Commit

Scroll down and click Commit changes. Your post will go live in ~1 minute.

Step 5: Add a Card to Blog List

  1. Edit posts.html.
  2. Add a card like this inside the grid:
<div class="card">
  <h3><a href="posts/2025-09-07-your-title.html">Post Title</a></h3>
  <small>Your Name • 2025-09-07</small>
  <p>A short summary here.</p>
</div>

Step 6: Add Images (Optional)

  1. Go to the assets/ folder → Upload files.
  2. Insert images in your post like: 
    <img src="../assets/your-image.png" alt="Description">
